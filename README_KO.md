# SideClip - 크롬 클립보드 히스토리 확장 프로그램

브라우저의 사이드 패널에서 텍스트와 이미지를 모두 지원하는 강력한 클립보드 히스토리 기능을 제공하는 크롬 확장 프로그램입니다.

## ✨ 주요 기능

### 📋 텍스트 관리
- **자동 텍스트 캡처**: 복사한 텍스트를 자동으로 클립보드 히스토리에 저장
- **스마트 텍스트 미리보기**: 처음 100자 표시, 클릭 시 전체 내용 복사
- **텍스트 서식 보존**: 줄바꿈과 공백 유지
- **빠른 복사**: 원클릭으로 텍스트 항목을 클립보드로 복사

### 🖼️ 이미지 지원
- **이미지 캡처**: 우클릭 컨텍스트 메뉴로 이미지를 히스토리에 저장
- **썸네일 미리보기**: 반응형 크기 조정이 가능한 고품질 썸네일
- **포맷 변환**: 클립보드 호환성을 위한 자동 PNG 변환
- **크기 정보**: 파일 크기 및 해상도 표시
- **대용량 이미지 지원**: 최대 5MB 이미지 처리 및 자동 정리

### 🎨 사용자 인터페이스
- **사이드 패널 인터페이스**: 크롬 사이드 패널의 현대적이고 반응형 UI
- **시각적 구분**: 텍스트와 이미지 항목을 위한 명확한 아이콘 표시
- **반응형 디자인**: 다양한 패널 크기에 적응 (300px-400px+)
- **호버 효과**: 부드러운 애니메이션과 시각적 피드백
- **빛나는 테두리**: 이미지 썸네일의 우아한 호버 효과

### 🔧 고급 기능
- **영구 저장**: 이미지용 IndexedDB, 텍스트용 chrome.storage
- **실시간 업데이트**: 클립보드 변경 시 즉시 UI 업데이트
- **스마트 정리**: 자동 저장소 관리 및 할당량 처리
- **오류 복구**: 폴백 메커니즘을 통한 강력한 오류 처리
- **CORS 솔루션**: 크로스 오리진 이미지를 위한 백그라운드 스크립트 처리

### ⌨️ 키보드 및 제어
- **키보드 단축키**: Ctrl+Shift+H로 빠른 접근 (Mac에서는 Cmd+Shift+H)
- **삭제 관리**: 개별 항목 제거 또는 전체 히스토리 삭제
- **확인 대화상자**: 사용자 확인을 통한 안전한 삭제
- **컨텍스트 메뉴**: 클립보드 이모지가 포함된 향상된 우클릭 메뉴

## 📖 사용 가이드

### SideClip 열기

**방법 1**: 크롬 툴바의 SideClip 아이콘(📋) 클릭  
**방법 2**: 키보드 단축키 `Ctrl+Shift+H` 사용 (Mac에서는 `Cmd+Shift+H`)

### 텍스트 작업

1. **텍스트 복사**: 텍스트를 선택하고 `Ctrl+C` (또는 `Cmd+C`) 사용
2. **히스토리 보기**: 텍스트가 타임스탬프와 함께 SideClip에 자동으로 나타남
3. **텍스트 재사용**: 텍스트 항목을 클릭하여 클립보드로 다시 복사
4. **텍스트 삭제**: "×" 버튼을 사용하여 개별 항목 제거

### 이미지 작업

1. **이미지 저장**: 이미지 우클릭 → "📋 SideClip에 이미지 복사"
2. **이미지 보기**: 썸네일과 크기 정보가 포함된 이미지 표시
3. **이미지 복사**: 이미지 썸네일을 클릭하여 클립보드로 복사 (PNG 형식)
4. **이미지 세부정보**: 호버 시 해상도와 파일 크기 확인

### 관리 및 정리

1. **개별 삭제**: 항목 옆의 "×" 버튼 사용
2. **모두 삭제**: "모두 삭제" 버튼으로 전체 히스토리 제거 (확인 후)
3. **자동 정리**: 히스토리를 50개 항목으로 제한 (가장 오래된 항목 자동 제거)
4. **저장소 관리**: 5MB 이상 이미지 자동 건너뛰기

## ⌨️ 키보드 단축키 및 제어

- **`Ctrl+Shift+H`** - SideClip 사이드 패널 토글
- **`Escape`** - 확인 대화상자 닫기
- **클릭** - 항목을 클립보드로 복사
- **우클릭 + 컨텍스트 메뉴** - 이미지를 SideClip에 저장

## 🛠️ 기술 사양

### 핵심 기술
- **매니페스트 버전**: 3 (최신 크롬 확장 프로그램 표준)
- **저장소**: 이미지용 IndexedDB, 텍스트용 chrome.storage.local
- **권한**: storage, sidePanel, activeTab, contextMenus, scripting, clipboardRead
- **호스트 권한**: CORS 처리를 위한 모든 HTTP/HTTPS 사이트

### 성능 및 제한
- **히스토리 제한**: 최근 50개 항목
- **이미지 크기 제한**: 이미지당 최대 5MB
- **텍스트 미리보기**: 히스토리 목록에서 처음 100자
- **저장소 정리**: 자동 할당량 관리

### 브라우저 지원
- **크롬**: 버전 114+ (Manifest V3 요구사항)
- **사이드 패널 API**: 크롬 114+ 필요
- **IndexedDB**: 이미지 저장을 위한 완전한 브라우저 지원
